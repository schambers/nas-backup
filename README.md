# NAS Backup

Backup script to backup NAS to Google Drive. Uses [python-rclone](https://github.com/ddragosd/python-rclone) for the heavy lifting.

## Create Python venv

`python3 -m venv env`

## Activate venv

`source venv/bin/activate`

## Confirm venv

`which python` should print .../venv/bin/python
